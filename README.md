# Quick Start

1. Install Orderly
2. Launch the application
3. Select a folder
4. Preview the organization plan
5. Click **Run**

Undo is available after each run if you want to revert changes.

---

# What is Orderly

Orderly is a desktop file organization tool designed for predictable and transparent workflows.

Instead of automatically moving or deleting files in the background, Orderly analyzes your folder and shows a full preview before any changes happen.

Every operation is reversible.

---

# Core Workflow

1. Select a folder or profile
2. Preview the organization plan
3. Run when ready
4. Undo if needed

Everything is visible before execution.

---

# Key Features

## Preview-first organization

See exactly what will happen before any file operation.

Files are analyzed and categorized before execution.

---

## Instant undo

Every run can be reversed.

Undo restores the previous folder state.

---

## Profiles

Save reusable organization setups for common workflows.

Profiles allow recurring folder automation.

---

## Run history

Each execution is recorded.

You can review previous runs and open detailed reports.

---

## Activity timeline

Track the latest scan and actions performed on a profile.

---

# Automation

Orderly supports optional folder monitoring.

## Folder watching

Automatically monitor folders for new files.

## Auto-apply

Apply organization rules automatically when new files appear.

## Pause watching

Temporarily disable watchers from the tray menu.

---

# Windows Explorer Integration

Orderly integrates with the Windows 11 context menu.

Right-click any folder and access:

- Preview
- Run now
- Undo last run
- Watch (On / Off)
- Auto-apply (On / Off)
- Open Orderly
- Settings

This allows quick actions without opening the main application.

---

# System Tray Mode

Orderly runs in the background and stays available in the system tray.

Tray menu features include:

Orderly uses manual updates.

---

# Privacy

Orderly is designed with a **local-first architecture**.

Core file organization runs on your machine.

Optional features that may use network access include:

- update checks
- crash reporting
- manual diagnostics submission

Orderly does **not upload or sync your files** as part of normal organization workflows.
