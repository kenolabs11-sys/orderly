# Support

If you need help with Orderly, please follow the options below.

---

## Reporting a bug

If you believe you found a bug:

1. Check existing issues first
2. Open a new issue if the problem has not been reported
3. Include steps to reproduce the problem

Helpful information includes:

- Orderly version
- Windows version
- What you expected to happen
- What actually happened
- Screenshots if relevant

---

## Using the built-in diagnostics

Orderly includes diagnostic tools inside the application.

You can send a report directly from the app:

Settings → Advanced → Diagnostics → **Send report**

This will automatically include logs and diagnostic information.

You can also export a diagnostics ZIP file manually.

---

## Feature requests

If you have an idea to improve Orderly:

Open a feature request in the Issues section.

Please describe:

- the workflow you want to improve
- the problem you are trying to solve
- possible solutions

---

## General questions

For general usage questions, you can also open an issue.

---

## Before opening an issue

Please check:

- the documentation
- existing issues
- recent releases

Your question may already be answered.
